# MediaStack 管理界面设计方案

## 设计概览

这是一个为 MediaStack 直播录制管理系统设计的现代化、专业化管理界面。设计灵感来自专业媒体工作站和直播控制台，采用深色主题，强调实时性和专业感。

## 设计理念

**主题定位：现代专业媒体工作站**

- **视觉风格**：深色科技感 + 流媒体特色
- **色彩基调**：深邃的深蓝黑背景 + 紫红渐变点缀
- **交互体验**：流畅的动画过渡 + 清晰的视觉反馈
- **专业感**：等宽字体 + 技术感图标 + 实时状态指示

## 核心设计元素

### 1. 色彩系统

**主色调：**
- **Primary Red**: `#E94560` - 用于强调、警告、录制状态
- **Secondary Purple**: `#8B5CF6` - 用于渐变、次要强调
- **Background**: `#0a0e1a` - 深邃的深蓝黑色背景
- **Surface**: `rgba(15, 20, 35, 0.6)` - 半透明深蓝色表面
- **Text**: `#e4e7eb` - 柔和的浅灰色文字

**渐变应用：**
```css
linear-gradient(135deg, #E94560 0%, #8B5CF6 100%)
```
用于：Logo、主按钮、活动状态指示、进度条

### 2. 字体系统

**品牌字体：**
- **JetBrains Mono** - 等宽字体，用于品牌名称、代码、技术数据
- **Noto Sans SC** - 优化的中文字体，用于界面文字

**字体层级：**
- 品牌名称：20px, 700 weight
- 页面标题：24px, 600 weight
- 章节标题：20px, 600 weight
- 正文：14px, 500 weight
- 辅助文字：12px, 400 weight
- 标签/徽章：11px, 600 weight

### 3. 图标系统

所有菜单项都配备了自定义 SVG 图标：

- **直播源管理** - 广播波纹图标
- **录制任务** - 录制圆点图标
- **定时计划** - 时钟图标
- **录制管理** - 文件夹图标
- **视频管理** - 视频摄像机图标
- **文件上传** - 上传箭头图标
- **分类管理** - 网格图标
- **标签管理** - 标签图标
- **观看码管理** - 钥匙图标
- **系统状态** - 图表图标

## 关键组件设计

### 1. 侧边栏 (Sidebar)

**品牌区域：**
- 渐变色 Logo (红紫渐变)
- 品牌名称 "MediaStack" (渐变文字效果)
- 副标题 "Live Recording Studio"
- LIVE 状态指示器 (脉动红点 + 文字)

**导航菜单：**
- 三个分组：录制控制、内容管理、系统设置
- 每个菜单项包含图标 + 文字
- 悬停效果：背景高亮 + 左侧渐变条
- 激活状态：背景色 + 边框 + 左侧渐变条

**用户信息：**
- 渐变色头像
- 用户名 + 角色
- 悬停效果

### 2. 顶部栏 (Top Header)

- 左侧：当前页面标题（动态显示）
- 右侧：系统状态组件（CPU、内存、磁盘、带宽）
- 半透明背景 + 毛玻璃效果

### 3. 内容区域

**表格样式：**
- 半透明深色背景
- 毛玻璃效果
- 圆角边框
- 表头：浅色背景 + 大写字母 + 字间距
- 行悬停：红色半透明背景
- 斑马纹：极浅的白色背景

**按钮样式：**
- 主按钮：红紫渐变 + 阴影 + 悬停上浮
- 成功按钮：绿色半透明 + 边框
- 危险按钮：红色半透明 + 边框
- 圆角：8px

**标签样式：**
- 半透明背景
- 无边框
- 圆角：6px
- 颜色语义化（成功/信息/警告/错误）

### 4. 背景效果

**渐变光球：**
- 两个大型模糊渐变圆形
- 红色光球（左上）+ 紫色光球（右下）
- 缓慢浮动动画（20秒循环）

**网格图案：**
- 细微的网格线
- 红紫双色
- 径向渐变遮罩（中心亮，边缘暗）

## 动画与交互

### 1. 页面过渡

```css
.page-fade-enter-active,
.page-fade-leave-active {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

- 进入：从下方淡入上移
- 离开：向上方淡出上移

### 2. 悬停效果

- 导航项：背景色变化 + 左侧渐变条伸展
- 按钮：上浮 2px + 阴影增强
- 表格行：背景色高亮

### 3. 状态动画

- LIVE 指示器：脉动动画（2秒循环）
- 背景光球：浮动动画（20秒循环）
- 加载状态：红色旋转圆环

## 响应式设计

当前设计针对桌面端优化（1920x1080+），建议后续添加：

- 平板适配（768px - 1024px）：侧边栏可折叠
- 移动端适配（< 768px）：底部导航栏

## 技术实现

**核心技术：**
- Vue 3 Composition API
- Element Plus UI 组件库
- CSS3 动画与过渡
- SVG 图标
- Google Fonts (JetBrains Mono + Noto Sans SC)

**性能优化：**
- 使用 `backdrop-filter` 实现毛玻璃效果
- CSS 动画优先于 JavaScript 动画
- 合理使用 `will-change` 属性
- 图标使用 SVG 而非图片

## 设计亮点

1. **品牌识别度强**：独特的红紫渐变 Logo + 等宽字体品牌名
2. **专业感十足**：深色主题 + 技术感图标 + 实时状态指示
3. **视觉层次清晰**：毛玻璃效果 + 渐变光球 + 网格背景
4. **交互流畅**：平滑的过渡动画 + 清晰的悬停反馈
5. **细节精致**：自定义滚动条 + 圆角设计 + 阴影层次
