设计亮点

  • 独特的品牌识别 - 红紫渐变 Logo + 等宽字体品牌名，避免常见的 AI 生成风格
  • 专业媒体工作站美学 - 深色主题 + 技术感图标 + 实时状态指示
  • 毛玻璃效果 - 使用 backdrop-filter 实现半透明模糊背景
  • 动态背景 - 渐变光球浮动动画 + 网格图案，增加视觉深度
  • 流畅的交互 - 平滑的页面过渡 + 悬停效果 + 状态动画
  • 语义化色彩 - 成功/警告/错误状态使用不同颜色，一目了然
  • 细节精致 - 自定义滚动条 + 圆角设计 + 阴影层次

技术实现

核心技术栈

  • Vue 3 Composition API
  • Element Plus UI 组件库
  • CSS3 动画与过渡
  • SVG 图标系统
  • Google Fonts (JetBrains Mono + Noto Sans SC)

关键 CSS 特性

  • backdrop-filter: blur(20px) - 毛玻璃效果
  • linear-gradient(135deg, #E94560 0%, #8B5CF6 100%) - 红紫渐变
  • cubic-bezier(0.4, 0, 0.2, 1) - 平滑过渡曲线
  • @keyframes - 脉动、浮动动画
  • :deep() - Vue 深度选择器，覆盖 Element Plus 样式
